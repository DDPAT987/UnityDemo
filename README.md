# UnityDemo
基于 Unity 2022 独立开发的 2D 俯视角 Roguelite 生存游戏 Demo，核心玩法参考《吸血鬼幸存者》。玩家在开放场地对抗源源不断的敌人，通过击杀获取经验升级，在随机升级选项中构建独特 Build 组合。
核心实现：
模块化武器系统： 设计抽象基类 WeaponBase + 4 种子类派生（圆周发射、鼠标指向、鼠标 AOE、被动击杀爆炸与伤害光环），每种武器持有独立的强化修改器结构体，支持伤害倍率、冷却倍率、弹丸数、范围、弹速等多维度独立升级，武器间属性完全隔离。
敌人 AI 与动态难度： 实现 Seek 追踪算法、屏幕外环形区域生成、基于玩家等级的动态权重分层系统（低级敌人权重递减、高级逐步解锁），支持近战与远程两种敌人类型，配合受击反馈（闪白、击退、浮动伤害数字）。
双层 Buff 架构： 设计传统拾取型 Buff 与可解锁触发型 Buff 两套独立系统。后者支持 7 种触发条件（被动/击杀计数/每次击杀/受伤/低血量/升级/周期性）和 9 种效果类型（临时属性、永久属性、掉率调整、回复倍率、击杀回血、伤害减免等），具备堆叠、冷却与持续时间管理。
升级与 Build 构建： 实现非线性经验曲线、随机 3 选 1 升级面板（含过滤与加权逻辑），升级类型覆盖属性提升、武器解锁、武器强化、Buff 解锁四大类共 30+ 条目，全部通过 ScriptableObject 配置驱动，策划可脱离代码调整数值。
